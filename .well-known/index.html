NeighbourGuard - Well-Known Resources

This directory contains well-known resources for the NeighbourGuard emergency response platform.

Chrome DevTools Configuration

/appspecific/com.chrome.devtools.json

Configuration file for Chrome DevTools integration, enabling enhanced debugging features for the NeighbourGuard web application including source maps, performance profiling, and security analysis.

Environment: Development (Port 5000)
