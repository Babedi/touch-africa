🎛️ How to Apply

Option 1 - Add to main page: Include the enhanced contrast CSS file in your main HTML:

<link rel="stylesheet" href="/shared/enhanced-contrast.css">

Option 2 - Toggle on demand: Load it dynamically when users need better contrast:

document.head.appendChild(contrastStylesheet);

Option 3 - User preference: Save user preference and auto-apply on future visits.
